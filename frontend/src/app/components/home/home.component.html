Funcionalidades

drag_indicator
Arrastar e Soltar Clique e segure o mouse em cima de um item, arraste-o até o local devido e solte o botão do mouse. Faça isso para todos os itens.
rule
Validar Clique no botão Validar em qualquer momento para saber o resultado do seu jogo. Os itens errados ficarão em vermelho e uma mensagem irá mostrar os totais. Pare o mouse em cima de um item errado por 1 segundo que uma dica irá mostrar o local correto.
done_all
Resolver Clique no botão de Resolver para que todos os itens sejam colocados em seus devidos lugares.
refresh
Reiniciar Clique no botão de Reiniciar para começar o jogo novamente. Os Itens para arrastar serão arrumados de forma aleatória.
